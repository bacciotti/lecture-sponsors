# Bacciotti Lecture Sponsors Plugin

Just adds some company badges in sidebar.

## Requirements / Tested on
![Wordpress 5.8.1](https://img.shields.io/badge/Wordpress-5.8.1-green)  
![PHP 7.4](https://img.shields.io/badge/PHP-7.4-blue)

## Installation

Download the files, upload to wp-content/plugins and activate it.

## Usage
Go to Widgets page and select "Patrocinadores". You can select choose the checkbox you want to show.
